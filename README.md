# netflix-recommendation-system
Movie recommendation system using collaborative filtering and Python

## Project Overview
This project builds a movie recommendation system using collaborative filtering techniques.

The goal is to recommend movies to users based on their past ratings and similarities with other users.

## Technologies Used
Python
Pandas
NumPy
Scikit-learn

## Steps Performed
1. Data Cleaning
2. Exploratory Data Analysis
3. User–Item Matrix Creation
4. Collaborative Filtering Model
5. Recommendation Generation

## Outcome
The model predicts user preferences and suggests relevant movies based on historical rating patterns.
